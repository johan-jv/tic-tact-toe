Tic-Tac-Toe (Console Version)
--------------------------------
A classic Tic-Tac-Toe implementation with a dynamic twist:
When the board fills without a winner, the oldest 'X' and 'O' are removed,
allowing continuous gameplay.

Features:
- Player vs Player mode
- Player vs Computer mode (basic AI)
- Move history tracking (oldest moves removed)
- Infinite play loop (no draws)

--------------------------------
NOTE: Upcoming Version

This project is being extended into a new experimental game:

"Palimpsest Grid: Archivist Mode"

Key ideas in development:
- Temporal decay of moves (aging system)
- Ghost cells (residual board states)
- Soft turn restrictions (fairness balancing)
- Strategic time-based gameplay (beyond static Tic-Tac-Toe)

The upcoming version transforms the game from a simple grid strategy
into a temporal decision system where both position and timing matter.
